## Get Started

1. Clone the repository
```sh
> git clone https://github.com/ps3611/hoolihomes.git
```
For each directory *cron*, *server* and *client*, do:

2. Install dependencies
```sh
> npm install
```

3. Configure your environment

```sh
> cp .env.example .env
# Edit .env
```

4. Run the project

```sh
# Launch directory
> npm start
```
